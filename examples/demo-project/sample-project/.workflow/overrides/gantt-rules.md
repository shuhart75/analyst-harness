# Gantt Rules Override

- Generated gantt headers should not be edited manually.
- Put quarter holidays into `planning/<quarter>/gantt/closed-days.txt`.
- Put shared preamble blocks into `planning/<quarter>/gantt/preamble/common.puml`.
- Put view-specific preamble blocks into `planning/<quarter>/gantt/preamble/<view>.puml`.
- Keep feature lanes separated as `-- Feature Name --` in the aggregated gantt.
