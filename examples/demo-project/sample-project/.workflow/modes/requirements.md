# Mode: requirements

## Goal

Produce or update living requirement packs from current-system context and change requests.

## Main inputs

- `context/current-system/requirements/`
- `context/current-system/screenshots/`
- `context/current-system/prototypes/`
- `context/current-system/diagrams/`
- `context/change-requests/`

## Main outputs

- `features/*/references.md`
- `features/*/slices/*/slice.md`
- `features/*/slices/*/requirements/frontend.md`
- `features/*/slices/*/requirements/backend.md`

## Forbidden without mode switch

- changing agreed planning estimates
- changing quarter or commander baseline gantt
- changing actual execution dates
